# 오르리 AI동화 — 랜딩 페이지

> 40대 이상 개인사업자를 위한 실전 AI 동화 제작 과정 소개 & 상담 신청 랜딩 페이지

---

## 🎯 프로젝트 목표

- 오르리(엄인숙) 작가의 브랜드 소개 및 신뢰 구축
- 4050 세대 대상 AI 동화 제작 과정 안내
- 무료 상담 신청 접수 및 데이터 저장
- 카카오톡 채널 / 전화 상담 유도

| 구분 | 기능 |
|------|------|
| 헤더 | 고정 네비게이션, 스크롤 시 그림자 효과, 상담 신청 버튼 |
| 히어로 | 메인 카피, 파티클 애니메이션, 실적 통계(25권/38권/20개월/47세) |
| 고민 섹션 | 3가지 Pain Point 카드 |
| 방법 소개 | 핵심 도구(ChatGPT·Midjourney·Canva) 소개, 일러스트 영역 |
| 오르리 스토리 | 작가 프로필, 스토리 텍스트 |
| 혜택 섹션 | 3가지 변화 카드 |
| 커리큘럼 | STEP 01~04 단계별 안내 |
| FAQ | 아코디언 방식 5개 질문 |
| 상담 폼 | 성함·연락처·관심분야 입력, API 저장, 성공 모달 |
| 연락처 | 카카오톡 채널 링크, 전화번호 클릭 연결 |
| 모달 | Coming Soon 모달, 상담 완료 모달 |
| 플로팅 버튼 | 카카오톡(우측 하단), 맨위로(우측 하단), 배경음악(좌측 하단) |
| 배경음악 | 재생/정지 토글, 첫 클릭 자동 시작 |
| 반응형 | 모바일 / 태블릿 / 데스크톱 완전 대응 |
| 애니메이션 | 스크롤 페이드업, 파티클 효과, 네온 펄스 |

## 🗂️ 파일 구조

```
index.html          ← 메인 랜딩 페이지 (단일 파일)
illustration.png    ← (선택) AI 동화 일러스트 이미지
profile.png         ← (선택) 오르리 대표 프로필 사진
README.md
```

### 이미지 업로드 안내
- `illustration.png` : 히어로/방법소개 섹션 일러스트 (3:4 비율 권장)
- `profile.png` : 작가 프로필 사진 (정방형 권장)
- 이미지가 없어도 플레이스홀더가 표시되므로 문제없이 동작합니다.

## 📡 API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `tables/consultations` | 상담 신청 데이터 저장 |
| GET | `tables/consultations` | 상담 신청 목록 조회 |

### 저장 데이터 구조 (consultations 테이블)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동) |
| name | text | 신청자 성함 |
| phone | text | 연락처 |
| interest | text | 관심 분야 |
| submitted_at | datetime | 신청 일시 |
